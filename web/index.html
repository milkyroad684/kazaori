カゼオリ Kazaori

離散格子の風パズル。ファンを置いて、葉を渦穴へ導く。

ファン 0 / 0
向き

遊び方

  • 上の向きを選び、空セルをクリックするとファンが配置されます。
  • 配置済みファンをクリックすると撤去できます（向きを変えたいときは撤去してから置き直し）。
  • ファンは前方に 3 / 2 / 1 の強さで風を送ります。風はファン・壁・盤外で止まります。
  • 葉は毎ターン、立っているセルの合計風の主な向きへ 1 マス進みます。同じ強さの直交合成では止まります（干渉ロック）。
  • 渦穴(●)に入れば成功、茨(▲)に触れたら失敗です。

凡例: ■壁 ●渦穴 ▲茨 ▶固定ファン ▶配置ファン ●葉
